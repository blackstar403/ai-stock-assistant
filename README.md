## 🧙‍♂️ 交易模式建立：从混沌到系统

AlphaBot不仅仅是一个数据分析工具，更是你建立个人交易系统的导师：

### 🏗️ 交易系统构建器

- **交易风格识别**：通过分析你的历史交易，识别你是价值投资者、趋势追随者还是摸鱼高手
- **策略模板库**：提供多种经典交易策略模板，从简单的均线系统到复杂的多因子模型
- **个性化参数优化**：根据你的风险偏好和资金规模，优化交易参数
- **回测与验证**：在历史数据上验证你的交易系统，看看它是否真的有效（还是只是运气好）

### 📊 数据驱动决策

- **多维度分析**：技术面、基本面、资金面、情绪面全方位分析，不再单一维度看问题
- **概率思维培养**：展示交易的概率分布，帮助你理解"确定性"在市场中是不存在的
- **情景模拟**："如果明天大盘跌5%，你的持仓会怎样？"提前做好心理准备

## 🧘‍♀️ 交易纪律养成：从冲动到冷静

### 🚦 交易信号系统

- **客观入场提醒**：当市场符合你预设的条件时，发出提醒，而不是基于"感觉"

- **压力测试**：模拟极端市场情况，测试你的心理承受能力
- **冷静期设置**：在大幅亏损后强制执行交易冷静期，防止"追回损失"的冲动决策

## 🚀 快速开始

### 方法一：使用 Docker（推荐）

项目提供了 Docker Compose 配置和一键部署脚本，可以快速部署整个应用。

#### 前提条件

- Docker 和 Docker Compose 已安装
- 一颗平常心（这个很重要）

#### 部署步骤

1. 克隆仓库
```bash
git clone https://github.com/x-pai/ai-stock-assistant.git
cd ai-stock-assistant
```

2. 运行部署脚本
```bash
./deploy.sh
```

3. 访问应用
- 后端 API 文档：http://localhost:8000/api/docs
- 前端界面：http://localhost:3000

### 方法二：手动安装（适合喜欢折腾的朋友）

#### 后端

```bash
cd backend
pip install -r requirements.txt
# 配置 .env 文件
python train_model.py  # 可选，训练机器学习模型
uvicorn app.main:app --reload
```

#### 前端

```bash
cd frontend
npm install
npm start
```
